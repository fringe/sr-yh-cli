# 永辉数据工具
 yh用来处理永辉数据的工具的集合

# Features
 yh用来做数据转换，将各种源数据转换为目标数据

 # Quick Start

 `npm install sr-yh-cli -g`

 ```
  Usage: conv [options]

  将永辉的数据格式由json转为markdown

  Options:
    -i, --input <input>    输入文件名
    -o, --output <output>  输出文件名
    -u, --unique           根据event对数据进行去重
  -h, --help             output usage information
 ```
